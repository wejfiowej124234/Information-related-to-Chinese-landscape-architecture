# Information related to Chinese landscape architecture

本仓库用于整理“园林绿化公司资质/证照/投标材料”的 Markdown 文档库（含 Mermaid 图表），强调：

- 可读、可维护：入口/清单/速查/图表/打包表形成闭环导航
- 可对外：统一脱敏，遵循“最小披露”原则
- 可自动化：提供脚本用于批量改名、移动分组、修复链接、断链检查
